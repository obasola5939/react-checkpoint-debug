BUG 4: Infinite Re-renders in Counter
Investigation:

Used React DevTools Profiler to detect performance issues

Found Counter component re-rendering excessively

Identified incorrect dependency arrays in useCallback

Fix:

jsx
// Updated to use functional updates and remove history from dependencies
const increment = useCallback(() => {
  setCount(prevCount => prevCount + 1);
  setHistory(prevHistory => [...prevHistory, count + 1]);
}, [count]); // Removed history from dependencies

const decrement = useCallback(() => {
  setCount(prevCount => prevCount - 1);
  setHistory(prevHistory => [...prevHistory, count - 1]);
}, [count]); // Removed history from dependencies
